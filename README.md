# Pizza Delivery 3D

A 3D hyper-casual game where players deliver pizzas while navigating through obstacles. Developed using C# and Unity.

## Introduction

Collect all pizzas and deliver them to as many customers as possible! More you collect, more points you get. If you can collect enough pizzas, you might satisfy the hunger of a special customer and get a big bonus!

Beware of obstacles and traps, they will make you drop some of your pizzas!

## Features

- Delicious pizza 🍕🍕🍕
- Multiple levels with increasing difficulty
- Various obstacles and traps
- Special customers with big bonuses
